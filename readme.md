# 🎓 Анализатор обратной связи выпускников (AI Graduate Feedback Analyzer)

Добро пожаловать! Это веб-приложение на базе искусственного интеллекта (CrewAI + Google Gemini 2.5), созданное для автоматического анализа отзывов выпускников образовательных программ.

Приложение помогает университетам, онлайн-школам и курсам понять, как именно их учебная программа влияет на реальную карьеру студентов, выявить слабые места и получить готовые рекомендации по улучшению.

## 🚀 Что делает это приложение?

Вместо того чтобы вручную читать сотни отзывов, вы просто загружаете их в систему. Под капотом работает команда из трех виртуальных AI-агентов:

1. **🕵️‍♂️ Тематический аналитик:** Читает все отзывы и разбивает их на категории (качество преподавания, актуальность материалов, инфраструктура).
2. **💼 Карьерный консультант:** Сопоставляет жалобы или похвалу с *текущей должностью* выпускника (например, понимает, почему нехватка определенных навыков критична именно для Backend-разработчиков).
3. **🎓 Проректор:** Анализирует выводы первых двух агентов и пишет для вас финальный, структурированный управленческий отчет в формате Markdown.

## 📋 Инструкция: Как пользоваться?

### Шаг 1: Подготовьте данные
Система принимает файлы в формате **CSV**. В вашем файле обязательно должны быть две колонки с точными названиями (на английском):
* `review` — сам текст отзыва выпускника.
* `job` — текущая должность или место работы выпускника.

**Пример правильного файла (grad_data.csv):**
| review | job |
| :--- | :--- |
| "Нам давали очень старую теорию, на работе пришлось переучиваться." | Junior Frontend Developer |
| "Отличная математическая база, сильно помогла на собеседованиях!" | Data Scientist |
| "В аудиториях постоянно ломались компьютеры..." | QA Engineer |

### Шаг 2: Загрузите файл
Откройте приложение по ссылке. В левой панели (Sidebar) нажмите кнопку **Browse files** и выберите ваш `.csv` файл. Как только файл загрузится, появится зеленое сообщение об успехе.

### Шаг 3: Запустите анализ
Нажмите большую кнопку **«Запустить анализ агентами»**. 

> ⚠️ **Важно:** Процесс занимает **от 2 до 4 минут**. Это не ошибка! Искусственный интеллект работает вдумчиво, делая паузы между запросами (чтобы не превышать лимиты серверов Google) и передавая данные от одного агента к другому. Пожалуйста, не закрывайте вкладку и не обновляйте страницу, пока крутится индикатор загрузки.

### Шаг 4: Получите отчет
Как только агенты закончат работу, на экране появится подробный управленческий отчет: с сильными сторонами программы, критическими слабостями и конкретными рекомендациями для руководства. Вы можете скопировать его и использовать в работе.
