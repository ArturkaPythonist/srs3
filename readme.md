# СРС №2: Виртуальная симуляция дебатов ученого совета 🎓

**Вариант:** 13
**Выполнил:** Артур Шакиев
**Дисциплина:** Программная реализация алгоритмов взаимодействия и обмена данными

## Описание проекта
Данный проект представляет собой веб-приложение на базе мультиагентной системы (МАС). Система симулирует интеллектуальную дискуссию (дебаты) в рамках ученого совета. 

Взаимодействие построено на базе двух AI-агентов:
1. **Докладчик (Researcher):** Генерирует аргументированный научный доклад в защиту предложенного тезиса.
2. **Оппонент (Critic):** Анализирует доклад, выявляет логические уязвимости и выносит финальный вердикт (Одобрено/Отклонено).

## Технологический стек
* **Язык:** Python 3.12
* **Фреймворк МАС:** [CrewAI](https://crewai.com/)
* **Интерфейс:** [Streamlit](https://streamlit.io/)
* **LLM:** OpenAI API (gpt-4o-mini / gpt-3.5-turbo)

## Структура репозитория
* `streamlit_app.py` — основной код логики агентов и веб-интерфейса.
* `requirements.txt` — список зависимостей для корректного деплоя.
* `README.md` — документация проекта.
